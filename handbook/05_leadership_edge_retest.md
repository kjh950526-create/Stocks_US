# Handbook 05 — 리더십·엣지 재검증 (2012–2026, 다체제)

> 다른 세션에서 돌린 14년·5팩터·다체제(bull+bear) 재검증. 기존 STATUS/handbook 03의
> 리더십 결론을 **넓은 표본으로 다시 돌렸더니 방향성은 살아남았지만 크기는 과장이었다.**
> 이건 내 실제 매매 엣지 연구다(자동화/봇과는 분리). 재현 파라미터: RS_LB=63, FWD=40,
> STEP=10, SPY 200d SMA 레짐, 등가중, AdjClose. 유효표본 9,998 (bull 8,122 / bear 1,876).

## 팩터 5개 (등가중 total-return)
semi(NVDA·AVGO·AMD·MRVL·MU·QCOM·LRCX·AMAT·KLAC·ADI) / megatech(AAPL·MSFT·GOOGL·AMZN·
META·NFLX) / software(CRM·ADBE·NOW·INTU·WDAY·SNPS) / solar(FSLR·ENPH·SEDG·RUN) /
cyber(PANW·FTNT·ZS·CRWD·OKTA·S — 풀바스켓 ~2019~만, 다체제 커버 약함).

## 확인·강화된 것
1. **팩터 로테이션 = 강하게 확인.** 고정 리더 없음, 1~3년 주기 교체(2013 solar…2023 cyber…
   2025~26 semi). **2022 약세장엔 solar만 +22%, 나머지 −36~−52%** = 약세장엔 리딩 팩터가
   방어적으로 뒤집힘.
2. **리더십 = 지속 상태(신규 정량화).** P(리더 다음 리밸런스에도 유지) 58~74%, 평균 재임
   24~38거래일(solar 74%/38일 … software 58%/24일). → 리더는 5~8주 단위 유지, 매일 깜빡이는
   신호가 아님.
3. **동조성 확인 + bear에서 상승.** 팩터 내 평균 60d 상관 전체 0.46~0.56인데 **bear에서
   0.55~0.68로 오름**(semi/software/cyber bear 0.67~0.68). → 셀오프 때 팩터가 한 몸으로 갭 =
   분산 소멸이 **하필 갭 위험 최대일 때** 발생. §7 "동반상승→동반하락" 우려를 정량 확인.

## ★교정된 것 (소표본 AI강세장의 과장 — 중요)★
전체표본 fwd40: **rank1 6.5% / rank2 4.7% / laggard 4.7%.**
- **리더 우위 = ~1.8%p로 완만.** 기존 STATUS의 "+7.5~9%"는 **과장(semi+cyber 2020~26 AI강세장
  소표본 아티팩트).**
- **"rank2가 최악" = 기각.** rank2 ≈ laggard(둘 다 4.7).
- **"+40% 몬스터는 리더에서만" = 틀림.** 단일 최대러너(+182%)는 **laggard**에서 나옴. 리더는
  P(>40%) 5% vs 3%로 tail이 약간 두꺼울 뿐, 독점 아님.

## ★가장 중요한 신규 뉘앙스 — 리더 엣지는 팩터마다 다르다★
| 팩터 | 리더우위(r1−lag) |
|---|---|
| solar | **+5.0** |
| semi | **+4.4** |
| megatech | +1.7 |
| cyber | +1.3 |
| software | **−2.3** (laggard가 리더를 이김) |

→ 리더 엣지는 **고분산·고베타 팩터(solar/semi)에서 크고, software에선 음수.** 기존 연구가
semi 중심이라 엣지가 커 보였던 것. **결론 정교화: "리더를 사되, 팩터 성격(분산도/베타)이 리더
엣지 크기를 좌우한다."** 고베타 팩터일수록 리더 집중이 값어치 있고, 저분산 팩터(software)에선
리더 고집의 의미가 약하거나 역전.

## 실전 함의 (내 플레이북에 주는 것)
- **리더를 산다는 원칙은 유지** — 방향성은 다체제서 살아남음(모든 레짐서 rank1이 1등).
- 근데 **"리더가 팻테일 독점"이라는 강한 서사는 폐기.** 리더 우위는 완만(~1.8%p)하고 팩터의존적.
- **팩터 선택이 종목 선택만큼 중요해짐** — 고베타(semi/solar)에선 리더집중이 유효, software류엔
  약함. 지금 내 유니버스(semi·cyber·fintech·defense) 중 **semi가 리더엣지 최대, cyber는 +1.3로
  약함**(그래서 cyber에선 리더 PANW보다 차트 좋은 FTNT를 봐도 되는 근거가 됨).
- **약세장 상관 상승(0.67+) = 팩터캡(갭 총량)이 가장 중요할 때가 셀오프** = §7 갭캡 영구 유지 재확인.

## 부록 — DAVE 케이스: "리더가 팩터를 끌고 간다"의 실증 + 초입 포착 방법론 (2026-07-16)

**관찰(Jeonghun):** DAVE는 2024-07~2026-03 약 1년간 못 뚫고 응집상승-급락 반복 → 최근 1~2달에
횡보 돌파. 왜 1년을 못 뚫었나?

**데이터(DAVE 월별 vs 핀테크 팩터 RS·순위):**
- 못 뚫던 이유 = **① 팩터 역풍(주원인)**: 2025 하반기~2026초 핀테크 팩터가 1위→4위(꼴찌) 추락,
  RS +72%→−31%. 팩터 죽으니 DAVE도 전고 대비 −39%. **팩터가 종목을 끌고 간다 실증.**
  **② 자체 급등 소화**: 그 전 2024-11(+150%/월)·2025-05(+100%/월) 극단 수직폭등의 소화(고ADR
  소형주 특성).
- ★결정적 반전: **최근 돌파(2026-04~06)는 팩터가 살아나서가 아니라 DAVE가 팩터보다 먼저 뚫음.**
  DAVE 신고 갱신 시점(2026-04)에 핀테크 팩터는 여전히 3/4위(RS −8%). DAVE가 먼저 튀어 팩터 RS를
  −8%→+52%로 끌어올림. = **리더가 팩터를 끌고 간다 / co-rest OFF(리더가 팩터에서 떨어져 나옴)가
  신호**의 실제 사례.

**→ 방법론(오늘의 수확, Jeonghun 교정판): 초입 신호는 "첫 돌파"가 아니라 "돌파 후 지속"이다.**
- ❌ 부정확: "리더의 첫 돌파 = 초입 신호" — 실시간엔 가짜와 구분 안 됨. DAVE는 그 전에도 여러 번
  "응집상승→뚫는 듯→폭락→수평횡보"를 반복(2024-11, 2025-05). 4월 돌파도 당시엔 "또 한 번의 가짜"로
  보이는 게 정상. 첫 돌파 하나로는 진짜/가짜 구분 불가.
- ✅ 정확: **리더가 돌파 후 "무너지지 않고 EMA 정배열을 유지한 채 EMA surf를 시작"하는 것**이 진짜
  초입 신호. DAVE는 5월말~6월에 뚫고 나온 뒤 안 떨어지고 정배열 유지 + EMA surf 상승채널 → 이게
  이전 가짜 돌파들과 결정적으로 다른 점. 이 "돌파 후 지속(수 주)"이 진짜/가짜 필터.
- **이건 §2 갈래A와 동일:** 나쁜구간 탈출→50/200 재탈환→EMA 정배열 시작→첫 응집이 진입. DAVE
  5~6월이 정확히 그것.
- **실전 타임라인:** 4월 돌파=관심시작(가짜일수 있음) → 5~6월 정배열유지·안무너짐="이번엔 진짜"
  확인=진입근거 형성 → 6월 EMA surf=진행 → 7월 첫 플래그=진입 자리. "②단계(돌파후 지속 확인)"를
  팩터 초입 신호로 인식하면 ④응집에서 주저없이 진입 가능.
- **핀테크는 특히 리더(강손) 구분이 결정적** — DAVE 혼자 끌고 감(RS 97 vs 2위 39). 약손(AFRM/
  HOOD) 무의미. cyber(리더엣지 얇음)와 정반대.
- 팩터 역풍일 땐 리더도 눌린다(DAVE −39%) → **팩터 국면 확인이 종목보다 먼저.** 단 바닥 탈출은
  리더가 먼저 하므로, 리더의 첫 돌파를 팩터 초입 신호로 감시.

**주의(방법론 vs 후회 구분):** 이 방법론은 **다음(아직 안 온) 로테이션에 쓰는 도구**다. DAVE의
초입(2026-04)은 이미 지났고 현재가($400~450)는 중반이다. 지나간 리더를 "샀어야"로 후회하는 데
쓰면 도구가 오용된다 — "다음 기회 포착"에 쓴다.
